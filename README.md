# AlphaMapleGuild

메이플스토리 길드원 참여 현황(주간미션·지하 수로·플래그 레이스)을 화면에서 자동으로 읽어 `result.xlsx`로 정리하는 도구입니다.

## 사용 방법
1. 메이플 길드 창에서 **길드원 참여 현황(컨텐츠 참여 현황 탭)** 화면을 엽니다.
    * 길드 창을 확대/축소하지 말고, 위치를 고정한 채 사용하세요.
    * (대안) 전체 화면 캡처 이미지를 확대/축소 없이 띄워 두고 넘겨도 됩니다.
1. 프로그램을 실행합니다.
    * `.exe` 또는 `.py` 모두 실행 가능합니다. (`.py`는 `requirements.txt` 참고)
      * `.exe`는 옆에 **`data` 폴더가 같이 있어야** 합니다.
    * 대중적으로 사용된 적 없는 프로그램이라 백신이 오탐(바이러스로 탐지)하거나 실행을 차단할 수 있습니다.
1. 프로그램이 길드 창 위치를 자동으로 찾습니다. 이후 **스크롤 위치를 맞춘 뒤 `[Enter]`를 눌러 현재 화면을 기록**합니다.
    * 화면이 **멈춘 상태**에서 Enter를 누르세요. 한 화면에 보이는 행(약 19명)이 한 번에 기록됩니다.
    * 스크롤 → Enter를 반복해 모든 페이지를 기록합니다. 페이지가 겹쳐도 중복은 자동으로 제거됩니다.
    * 위/아래로 잘려 일부만 보이는 행은 자동으로 제외되니, 다음 Enter에서 온전히 보이게 스크롤하면 됩니다.
    * 다 기록했으면 **`[Q]`를 입력해 종료**하면 `result.xlsx`가 저장됩니다.
1. (최초 1회) `result.xlsx`의 **[닉네임 정보]** 시트에서 닉네임 이미지에 해당하는 **닉네임**을, 필요 시 **대표캐릭**을 입력해 줍니다.
   * OCR 기능이 없어 닉네임을 자동으로 문자 변환할 수 없으므로 수동 입력이 필요합니다.
   * 한 번 입력해 두면 이후 실행부터는 자동으로 닉네임 필드가 채워집니다.

## 접속 중 캐릭터 / 대표캐릭 매핑
* 접속 중인 길드원은 닉네임이 **흰색**으로 바뀌고 뒤에 `(대표캐릭)`이 붙어 표시됩니다. (오프라인은 **회색**, 대표캐릭만 표시)
* 한 계정이 부캐 여러 개를 함께 가입시킨 경우, 그때 접속한 캐릭에 따라 같은 계정이 서로 다른 형태로 보입니다.
* 이들을 한 사람으로 묶으려면 [닉네임 정보] 시트의 **[대표캐릭]** 칼럼에 대표 닉네임을 적어 주세요. 기록 시트에서는 대표캐릭으로 표시됩니다. (대표캐릭이 비어 있으면 닉네임으로 표시)

## 주의사항
* 프로그램 실행 중에는 길드 창의 위치를 바꾸거나 가려서는 안 됩니다.
* `[Enter]`를 누르는 순간, 닉네임·주간미션·지하 수로·플래그 레이스 값이 마우스 커서 등으로 가려지지 않게 하세요.
  * 닉네임이 가려지면 다른 사람으로 잘못 기록될 수 있습니다. (가린 채 눌렀다면 가리지 않고 다시 한 번 Enter)
  * 주간미션·지하 수로·플래그 레이스 숫자가 가려지면 해당 값은 `0`으로 인식됩니다.
* 직업·레벨·직위 칸 위에 마우스 커서를 두거나, 우측 스크롤 바로 스크롤하면 값 가림을 피하기 쉽습니다.
* 클릭으로 행이 **하늘색으로 강조**되거나, 온/오프라인에 따른 글자색 변화는 점수 판독에 영향을 주지 않습니다.
* 스캔 종료 후, 기존에 기록돼 있던 닉네임이 누락되었거나 새 닉네임이 추가된 경우 안내 메시지가 출력됩니다.

## 면책 조항

* 이 프로그램은 개인적으로 작성되었으며, 특정 단체의 승인, 사용 허가 등을 받지 않았습니다.
* 이 프로그램은 ***있는 그대로*** 제공됩니다. 프로그램 사용에 대한 책임은 각 사용자에게 있습니다.
